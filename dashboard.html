Contribuer

Ajouter une page : ouvrez ce fichier, une ligne dans CONFIG.pages suffit — .

Ajouter une page en 30 secondes

Tout se passe dans le bloc CONFIG, tout en haut de ce fichier HTML.

1 · Déposer le fichier

Placez votre page (ex. ma-page.html) dans le même dossier que ce fichier.

2 · Déclarer la page

pages: [
  { titre: "Mon titre", fichier: "ma-page.html" },   // ← minimum vital

  { titre: "Titre",                                  // version complète
    fichier: "autre-page.html",
    groupe: "Recours",              // crée / rejoint un groupe du menu
    description: "Sous-titre.",     // affiché dans l'index
    tags: "justice greco" },        // mots-clés pour la recherche
],

3 · Annexes & vidéo

annexes: [
  { titre: "Rapport X", url: "docs/rapport.pdf" },
  { titre: "Site Y", url: "https://…", ouvrir: "onglet" },
],
video: { titre: "…", url: "https://www.youtube.com/embed/VIDEO_ID" },
